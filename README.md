# AI 互动塔罗游戏网站开发 Prompt 文档（用于 bolt.new / AI Code Builder）

## 🎯 项目目标

开发一个 **纯前端、强视觉冲击、AI 手势交互的趣味网页游戏**。

风格参考：
https://gitee.com/ricky-ai/tarot

但主题改为：

👉 「蔡徐坤梗文化 + 3D塔罗抽卡 + 摄像头手势控制」

核心体验：

* 打开网页 → 摄像头启动
* 用户用手势隔空操作
* 3D 卡牌旋转/翻牌/洗牌
* 抽出“坤式塔罗结果”
* 伴随粒子、发光、BGM、表情包特效

整个过程：
❌ 不使用鼠标点击
✅ 完全手势交互

---

# 🧱 技术栈（必须严格遵守）

使用：

* React + Vite
* TypeScript
* TailwindCSS
* Three.js（或 CSS 3D Transform）
* MediaPipe Hands（手势识别）
* WebGL 粒子特效
* 无后端（纯静态站点）

禁止：

* 复杂服务器
* 登录系统
* 数据库存储

必须：

* 单页 SPA
* 可直接部署到 Vercel/Netlify
* 移动端 + 桌面端自适应

# 🃏 游戏流程设计

## Step 1 进入页面

功能：

* 全屏粒子背景
* 摄像头自动开启
* 显示手势识别框
* 提示文字：
  “举起你的手开始抽坤塔罗”

---

## Step 2 洗牌动画

效果：

* 多张 3D 卡牌在空间旋转
* 卡牌围绕中心悬浮
* 发光粒子轨迹
* 卡牌正面是蔡徐坤梗图

动画要求：

* 3D 透视
* 旋转 + 浮动
* hover 发光（由手势代替）

---

## Step 3 手势控制规则（非常重要）

使用 MediaPipe Hands：

### 手势 → 行为

| 手势    | 功能   |
| ----- | ---- |
| 张开手掌  | 进入游戏 |
| 握拳    | 抽牌   |
| 左滑    | 切换牌  |
| 右滑    | 切换牌  |
| OK 手势 | 翻牌   |
| 举起两只手 | 重新洗牌 |

要求：

* 实时识别
* 平滑过渡
* 无明显延迟

---

## Step 4 抽牌结果展示

翻牌动画：

* 卡牌旋转 180°
* 闪光
* 粒子爆炸特效
* 音效触发

结果内容示例：

* 「今日运势：唱跳Rap篮球大成功」
* 「建议：多练两年半」
* 「关键词：鸡你太美」
* 配对应蔡徐坤表情包
